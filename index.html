Quiénes somos

Un proyecto nacido
de la academia

Legalia es una plataforma de gestión legal desarrollada por estudiantes de Ingeniería de Software de la Universidad Libre de Colombia como proyecto integrador del semestre 2026-1.

Nuestro objetivo es democratizar el acceso a la justicia, conectando ciudadanos con profesionales del derecho de forma simple, segura y transparente — aplicando tecnología moderna al servicio del bien común.

Proceso

Tu proceso legal,
simplificado

Desde el primer contacto hasta el cierre del caso, Legalia organiza cada etapa de tu proceso jurídico de forma clara y accesible.

01

Regístrate y verifica tu perfil

Clientes crean su cuenta al instante. Abogados pasan por verificación de tarjeta profesional y especialidad antes de activar su cuenta.

02

Abre o recibe un caso legal

El cliente describe su situación, adjunta documentos y selecciona o recibe asignación automática de un abogado según especialidad.

03

Comunícate y gestiona en tiempo real

Chat seguro, notificaciones automáticas, agenda de citas y recordatorios de plazos procesales mantienen todo sincronizado.

04

Cierre y facturación automática

Al finalizar, se genera la factura digital, el cliente califica al abogado y el historial queda guardado permanentemente.

Perfiles de usuario

Diseñado para cada
parte del proceso

Cliente

Para quien necesita asesoría

  • Registrarse y crear perfil personal
  • Buscar abogados por especialidad
  • Abrir y hacer seguimiento de casos
  • Comunicarse directamente con su abogado
  • Programar citas y gestionar documentos
  • Realizar pagos y consultar facturas
  • Calificar al abogado al cerrar el caso
Abogado

Para quien brinda el servicio

  • Registrarse con verificación de credenciales
  • Gestionar portafolio de casos activos
  • Actualizar el estado y avances del proceso
  • Comunicarse de forma segura con clientes
  • Gestionar agenda y disponibilidad de citas
  • Subir y organizar documentos del expediente
  • Generar reportes detallados por caso
Administrador

Para quien gestiona la plataforma

  • Verificar y activar cuentas de abogados
  • Supervisar todos los casos registrados
  • Gestionar usuarios y controlar accesos
  • Generar estadísticas de uso del sistema
  • Monitorear transacciones y facturación
  • Administrar reportes generales
Tecnología

Stack tecnológico
robusto y moderno

Arquitectura de tres capas: presentación, lógica de negocio y datos. Escalable, segura y mantenible.

Capa 01
Presentación

HTML, CSS y JavaScript para la interfaz de usuario. Diseño responsivo y accesible para todos los dispositivos.

Capa 02
Lógica de Negocio

Python con Flask para las reglas de negocio, autenticación, procesamiento de pagos y lógica de casos.

Capa 03
Datos

PostgreSQL o MySQL para almacenamiento seguro, persistente y escalable de toda la información legal.

Funcionales

Verifican que cada RF se cumpla correctamente según los casos de uso documentados.

Usabilidad

Evalúan la experiencia del usuario final con la interfaz y el flujo de navegación.

Seguridad

Verifican autenticación, control de acceso y protección de datos sensibles.

Rendimiento

Pruebas de carga, estrés y estabilidad del sistema bajo condiciones de uso real.

Integración

Verifican la comunicación correcta entre módulos, APIs y la pasarela de pagos.

Aceptación (UAT)

Validación final con usuarios reales para confirmar que el sistema cumple expectativas.
